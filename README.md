# ESP8266 Web Relay Controller

Sistema de automatización IoT para controlar luces y dispositivos eléctricos mediante WiFi utilizando un ESP8266.

## Características
- Control de relés desde navegador web
- Interfaz HTML integrada en el ESP8266
- Programación de horarios automáticos
- Sincronización de hora mediante NTP
- Guardado de configuración en EEPROM
- Actualización remota OTA

## Tecnologías utilizadas
- ESP8266
- Arduino IDE
- C++
- HTML
- JavaScript
- WiFi

## Hardware utilizado
- Wemos D1 Mini / ESP8266
- Módulo de relés
- Fuente de alimentación
- Cableado
