# ESP8266 Web Relay Controller

Sistema IoT para controlar luces mediante WiFi usando un ESP8266.

Características:
- Control de 4 relés desde una interfaz web
- Configuración de horarios automáticos
- Sincronización de hora mediante NTP
- Guardado de configuración en EEPROM
- Actualización remota OTA
- Interfaz web optimizada para celular

Tecnologías utilizadas:
- ESP8266
- C++
- HTML
- JavaScript
- WiFi
- NTP
